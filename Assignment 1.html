This webpage enliststs the different formatting tags

    The different formatting tags used in HTML are as follows:
  1. bold text
  2. Important text
  3. Italics text
  4. Emphasized text
  5. marked text
  6. small text
  7. del text
  8. ins text
  9. sub text
  10. sup text

Description

Bold text
It is used to make the letters, words or sentecnces bold.For example:Hello World!.
Important text
It is used to highlight important text.For example:Hello World!
Italics text
It is used to make the letters, words or sentecnces slanting.For example:Hello World!
Emphasized text
It is used to emphasize text.For example:Hello World!
Marked text
It is used to mark text.For example:Hello World!
Small text
It is used to minimise the text.For example:Hello World!
Del text
It is used to delete or strike the text.For example:Hello World!
Ins text
It is used to insert text.For example:Hello World!
Sub text
It is used to type text as a subscript.For example:Hello World!
Sup text
It is used to type text as a superscript.For example:Hello World!
